# build

```
$ make fun
```

# quick run

1 . run

```
$ ./_bin/fun --code=110011 --start="2010-04-30" --end="2020-11-30" --money=10000
```

2 . Then find the generated file ( coded in .xls ) in xls dir.

3 . Have fun

# cli params

```
Usage of ./fun:
  -code string
    	code
  -debug
    	debug
  -end string
    	YYYY-MM-DD
  -frq int
    	frq (default 1)
  -money float
    	money (default 100)
  -rate float
    	rate (default 0.15)
  -start string
    	YYYY-MM-DD
```
